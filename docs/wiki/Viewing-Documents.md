# 문서 보기·이동

## GitHub 스타일 렌더링
- GFM 지원: 표, 체크박스, 취소선, 자동링크
- **언어별 코드 구문 강조**(highlight.js, 테마와 동기화)
- 라이트/다크 테마(툴바 🌙/☀️), 해 아이콘은 노란색
- 상대 경로 이미지 표시, YAML frontmatter는 숨김 처리

> 📷 _스크린샷: 표/코드블록/체크박스 렌더링_ — `images/view-render.png`

## 목차(ToC)
- 우측 패널에 문서 헤딩이 자동 추출됩니다. 클릭 시 해당 위치로 이동(이동 기록에도 반영).
- 우측 바 아이콘으로 목차 패널 숨김/표시, 경계 드래그로 너비 조절.

## 문서 간/내 링크 이동
- **문서 내 앵커**(`#제목`) 클릭 → 해당 섹션으로.
- **다른 문서 링크**(`other.md`, `docs/guide.md#섹션`) 클릭 → 같은 저장소 내 문서로 이동.
- **외부 링크**(http/https) → 기본 브라우저로 열림.

## 이동 기록(History)
상단 바에서 방문 경로를 **파일#앵커** 단위로 관리합니다.

> 📷 _스크린샷: 상단 이동 기록 바(브레드크럼 + 전체 기록)_ — `images/view-history.png`

- **← / →** 뒤로/앞으로
- **브레드크럼**(최대 3개): 같은 파일 내 이동은 `#앵커`, 다른 파일은 `파일명#앵커`로 표시
- **▾ 기록** 드롭다운: 파일별로 묶인 전체 기록 → 한 번에 점프
- **뒤로/앞으로 시 이전 스크롤 위치를 그대로 복원**합니다(보던 위치 유지).

## 일반텍스트 / 코드 파일
- `.txt`, `.json`, `.css`, 코드 등 **비-마크다운 파일**은 고정 글꼴의 일반텍스트 뷰어로 표시됩니다.
- 글꼴/기본 크기는 [환경설정](Preferences)에서 지정.
- 본문에서 **`Ctrl/⌘ +` / `-`** 로 크기 조절, **`Ctrl/⌘ 0`** 으로 기본값 복원.

> 📷 _스크린샷: 일반텍스트 뷰어_ — `images/view-plaintext.png`

## 렌더 프로파일
- 툴바 **렌더링** 드롭다운에서 렌더링 방식을 선택할 수 있습니다(현재 GitHub).
- 추후 수식(KaTeX)·다이어그램(Mermaid) 등 프로파일이 추가될 예정입니다.
