# 탐색기 (Explorer)

좌측 액티비티 바의 **탐색기(폴더)** 아이콘으로 엽니다. 등록한 **여러 폴더/저장소가 접이식 루트**로 나열됩니다.

> 📷 _스크린샷: 다중 루트 탐색기(로컬 폴더 + GitHub 저장소)_ — `images/explorer-roots.png`

## 루트(저장소/폴더)
- 루트 행은 **배경색으로 구분**되며 기본 **접힘** 상태입니다(최초 등록/실행 시).
- **▸/▾** 캐럿으로 펼치면 하위 트리를 지연 로드합니다.
- 펼침 상태는 저장되어 **탭 전환/재실행 후에도 유지**됩니다.
- 루트 우측 **🔄 갱신** 버튼: 해당 저장소 트리만 새로고침.

## 폴더 / 파일
- 폴더(📁/📂)와 파일(📄 마크다운 / 📃 일반) 아이콘으로 구분됩니다.
- **폴더 클릭** = 펼침/접힘만(우측에 열린 문서는 그대로 유지).
- **파일 클릭** = 해당 저장소를 활성화하고 본문 표시. 비-마크다운은 일반텍스트 뷰어로 열림.

## 우클릭 컨텍스트 메뉴
저장소명/폴더/파일에서 **우클릭** 하면 저장소별 옵션이 나옵니다.

> 📷 _스크린샷: 우클릭 컨텍스트 메뉴_ — `images/explorer-context.png`

- **저장소 갱신** — 트리 새로고침
- **온라인(브라우저)에서 보기 / 폴더 열기** — GitHub는 저장소 페이지, 로컬은 파일 관리자
- **제거(닫기)** — 탐색기에서 해당 루트 제거
- **파일 보기** — 이 저장소만의 표시 필터(전체/마크다운/일반텍스트/HTML·CSS)

## 표시 파일 필터
- 기본은 **마크다운만** 표시(폴더는 항상 표시).
- 전역 변경: 툴바 **⚙️ 환경설정 → 보여질 파일** (모든 저장소에 일괄 적용)
- 저장소별 변경: **우클릭 → 파일 보기** (해당 저장소만 override)
- 자세히 → [환경설정](Preferences)

## 패널 크기/배치
- 탐색기와 본문 사이 경계를 **드래그**해 너비 조절(저장됨).
- 좌측 아이콘으로 탐색기/GitHub 뷰 전환, 활성 아이콘 재클릭 시 숨김.
