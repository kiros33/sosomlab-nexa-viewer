# 환경설정 (Preferences)

툴바의 **⚙️** 버튼으로 별도 모달을 엽니다.

> 📷 _스크린샷: 환경설정 모달_ — `images/preferences.png`

## 탐색기에 보여질 파일
체크박스로 탐색기에 표시할 파일 종류를 선택합니다.

- **전체 파일** — 모든 파일 표시(선택 시 나머지 비활성)
- **마크다운** — `.md` `.markdown` `.mdx`
- **일반텍스트** — `.txt` `.log` `.csv` 등
- **HTML / CSS** — `.html` `.css` `.scss` 등

> ℹ️ 여기서 선택하면 **모든 폴더/저장소에 일괄 적용**됩니다.
> 저장소별로 다르게 보려면 탐색기에서 **우클릭 → 파일 보기**로 개별 설정(override)할 수 있습니다.
> (폴더는 필터와 무관하게 항상 표시됩니다.)

## 일반텍스트 / 코드 글꼴
비-마크다운 파일을 표시할 때 사용하는 글꼴을 지정합니다.

- **글꼴(font-family)** — 예: `ui-monospace, Menlo, Consolas, monospace`
- **기본 크기(px)** — 예: 14

> 본문에서 **`Ctrl/⌘ +` / `-`** 로 크기를 즉시 조절할 수 있고, **`Ctrl/⌘ 0`** 으로 기본 크기로 복원합니다.

## 저장
모든 설정(테마, 패널 너비/표시, 필터, 글꼴, 최근 문서, 이동 기록 펼침 등)은 자동 저장되어
다음 실행 때 복원됩니다.
