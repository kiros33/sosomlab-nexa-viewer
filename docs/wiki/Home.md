# Nexa Markdown Viewer 사용 설명서

**로컬과 GitHub 등 저장소의 Markdown을 GitHub 스타일로 읽는 크로스 플랫폼 데스크톱 뷰어** (Tauri 2 + React). 제작: **SosomLab**.

<p align="center">
  <img src="https://raw.githubusercontent.com/kiros33/sosomlab-nexa-viewer/main/docs/assets/screenshot.svg" alt="UI 개요" width="100%" />
</p>

> 위 이미지는 UI 개요입니다. 각 기능의 실제 화면은 아래 문서에서 다룹니다.

## 핵심 기능 한눈에

| 기능 | 설명 | 문서 |
|------|------|------|
| 로컬/원격 문서 보기 | 폴더·GitHub 저장소의 `.md`를 GitHub 스타일로 렌더 | [문서 보기](Viewing-Documents) |
| 다중 루트 탐색기 | 여러 폴더/저장소를 접이식 루트로 관리 | [탐색기](Explorer) |
| GitHub 연동 | 공개는 로그인 없이, 비공개는 PAT(암호화 저장) | [GitHub 연동](GitHub-Integration) |
| 이동 기록·스크롤 복원 | 링크 따라가기 + 뒤로/앞으로(위치 그대로) | [문서 보기](Viewing-Documents) |
| 내보내기 | HTML / PDF | [내보내기](Exporting) |
| 환경설정 | 표시 파일 필터, 일반텍스트 글꼴 | [환경설정](Preferences) |
