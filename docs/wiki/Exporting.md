# 내보내기 (Export)

툴바의 **⬇ 내보내기** 버튼을 누르면 모달이 열리고, 큰 아이콘으로 형식을 선택합니다.

> 📷 _스크린샷: 내보내기 모달(HTML/PDF 선택)_ — `images/export-modal.png`

## HTML
- 현재 렌더된 본문을 **스타일이 포함된 단일 `.html` 파일**로 저장합니다.
- `github-markdown-css`와 코드 하이라이트 스타일이 인라인되어, 브라우저에서 그대로 보입니다.
- 저장 위치를 묻는 다이얼로그에서 경로를 지정합니다.

## PDF
- 웹뷰 **인쇄** 기능을 통해 저장합니다(인쇄 대화상자에서 **"PDF로 저장"** 선택).
- 인쇄 시 사이드바/툴바 등 UI는 제외되고 본문만 출력되도록 구성되어 있습니다.

> 📷 _스크린샷: 인쇄 → PDF로 저장_ — `images/export-pdf.png`

## 참고
- 내보내기 버튼은 문서가 열려 있을 때 활성화됩니다.
- 이미지는 data URL로 포함되어 HTML 단일 파일에 함께 저장됩니다.
- 향후 서버사이드 PDF, 다크 테마 내보내기 옵션 등이 로드맵에 있습니다.
